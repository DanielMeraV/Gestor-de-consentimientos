Resumen de la Ley Orgánica de Protección de Datos Personales

1. Objeto y Finalidad

Esta ley garantiza el ejercicio del derecho a la protección de datos personales, incluyendo el acceso, decisión y control sobre la información personal.

2. Ámbito de Aplicación

Se aplica al tratamiento de datos personales en cualquier soporte, automatizado o no, con excepciones como:
  • Actividades personales o domésticas
  • Datos anonimizados
  • Actividades periodísticas
  • Seguridad y defensa del Estado

3. Principios Fundamentales

  • Legalidad: Tratamiento conforme a la ley
  • Lealtad: Transparencia en el manejo de datos
  • Finalidad: Uso específico y declarado
  • Proporcionalidad: Datos adecuados y necesarios
  • Calidad: Datos exactos y actualizados
  • Confidencialidad: Garantía de secreto y seguridad

4. Derechos del Titular

  • Acceso: Conocer qué datos se tienen
  • Rectificación: Corregir datos inexactos
  • Eliminación: Solicitar borrado de datos
  • Oposición: Negarse al tratamiento
  • Portabilidad: Transferir datos a otro responsable

5. Categorías Especiales de Datos

Protección especial para datos:
  • Sensibles (salud, religión, orientación sexual)
  • De niños y adolescentes
  • Crediticios
  • Biométricos

6. Transferencias Internacionales

Se requieren garantías específicas para transferir datos fuera del país, asegurando nivel adecuado de protección.

7. Sanciones

Infracciones clasificadas en:
  • Leves
  • Graves
  • Muy graves
Con multas proporcionales al tipo de infracción.
